# Double.AI

מרכז פיננסי חכם לצרכן הישראלי - מחשבונים, יועץ AI, ספריית יועצים וניתוח מסמכים.

## פיתוח מקומי

```bash
npm install
npm run dev
```

## בנייה לפריסה

```bash
npm install
npm run build
```

הקבצים המוכנים יהיו בתיקיית `dist`.

## פריסה

האתר פרוס ב-Cloudflare Pages ומחובר לדומיין double.co.il.
